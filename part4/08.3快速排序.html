⚡ 快速排序 (挖坑法 + 递归栈) 408 核心算法

基于分治思想：一趟排序使基准元素到达最终位置，将数组一分为二并递归处理。

🔥 408 考研核心定则 (快速排序)：

  • 时间复杂度：最好/平均 O(n log n)，最坏 O(n²) (原本就有序)
  • 空间复杂度：取决于递归栈深度，最好 O(log n)，最坏 O(n)
  • 稳定性：不稳定 (Unstable) (跨越式交换)
  • 核心考点：每一趟排序后，必有一个元素(基准)到达最终位置。

int Partition(int A[], int low, int high) {
    int pivot = A[low]; // 将表首元素作为基准，形成"坑"
    while (low < high) {
        while (low < high && A[high] >= pivot) --high;
        A[low] = A[high]; // 比基准小的移到左端坑位，high成新坑
        while (low < high && A[low] <= pivot) ++low;
        A[high] = A[low]; // 比基准大的移到右端坑位，low成新坑
    }
    A[low] = pivot; // 基准元素放到最终坑位
    return low;     // 返回存放基准的最终位置
}
void QuickSort(int A[], int low, int high) {
    if (low < high) {
        int pivotpos = Partition(A, low, high);
        QuickSort(A, low, pivotpos - 1);
        QuickSort(A, pivotpos + 1, high);
    }
}

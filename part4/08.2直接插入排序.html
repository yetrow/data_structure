  • 时间复杂度：最好 O(n) (正序)，最坏 O(n²) (逆序)，平均 O(n²)
  • 空间复杂度：O(1) (仅需一个 temp 辅助变量)
  • 稳定性：稳定 (Stable) (大于才移动，等于不移动)
  • 适用性：适用于顺序表和单链表 (基本有序时效率极高)

InsertionSort.c 

void InsertSort(int A[], int n) {
    int i, j, temp;
    for (i = 1; i < n; i++) {
        if (A[i] < A[i - 1]) {  // 若 A[i] 小于其前驱，则需插入
            temp = A[i];        // 暂存哨兵/待插入元素
            for (j = i - 1; j >= 0 && A[j] > temp; --j) {
                A[j + 1] = A[j];  // 将大于 temp 的元素后移
            }
            A[j + 1] = temp;    // 在空出的位置插入原 A[i]
        }
    }
}

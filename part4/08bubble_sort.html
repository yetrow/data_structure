C 语言实现

void bubble_sort(int a[], int size)
{
    for (int i = 0; i < size - 1; i++)
    {
        for (int j = 0; j < size - i - 1; j++)
        {
            if (a[j] > a[j + 1])
            {
                int temp = a[j];
                a[j] = a[j + 1];
                a[j + 1] = temp;
            }
        }
    }
}

算法要点

  • 比较对象：冒泡排序每次只比较相邻的 a[j] 和 a[j + 1]。
  • 交换条件：升序排序时，如果左边比右边大，就交换它们。
  • 轮次边界：第 i 轮结束后，末尾 i + 1 个元素已经有序，内层循环可以少比较这些位置。
  • 时间复杂度：两层循环决定了平均和最坏时间复杂度都是 O(n^2)。

本页演示的关键修正

冒泡排序的交换条件应比较相邻元素：a[j] > a[j + 1]。 如果写成 a[j] < a[i]，比较对象就不是当前相邻对，排序过程会偏离冒泡排序的逻辑。

🔀 归并排序 (Merge Sort) 408 核心算法

分治法经典应用：将数组递归对半拆分，再借助辅助数组合并两个有序表。

🔥 408 考点速记：

  • 时间复杂度：最好、最坏、平均均为 O(n log n)。
  • 空间复杂度：O(n) (需要一个大小为 n 的辅助数组 B)。
  • 稳定性：稳定 (Stable)，取决于 B[i] <= B[j] 的判断。
  • 核心考点：“二路归并”形态构成倒立二叉树，趟数为 $\lceil \log_2 n \rceil$。

💻 归并排序 C语言实现 (王道标准) 

int *B; // 声明全局辅助数组
void Merge(int A[], int low, int mid, int high) {
    for(int k = low; k <= high; k++)
        B[k] = A[k]; // 将 A 中元素复制到 B 中
    
    int i = low, j = mid + 1, k = i;
    while(i <= mid && j <= high) {
        if(B[i] <= B[j])      // <= 保证了算法的稳定性
            A[k++] = B[i++];  // 左半段较小，填入A
        else
            A[k++] = B[j++];  // 右半段较小，填入A
    }
    while(i <= mid)  A[k++] = B[i++]; // 若左半段有剩余，全放入A
    while(j <= high) A[k++] = B[j++]; // 若右半段有剩余，全放入A
}

void MergeSort(int A[], int low, int high) {
    if(low < high) {
        int mid = (low + high) / 2;  // 从中间划分
        MergeSort(A, low, mid);      // 对左半部分归并排序
        MergeSort(A, mid + 1, high); // 对右半部分归并排序
        Merge(A, low, mid, high);    // 归并两个有序子序列
    }
}

🧮 计数排序 (含负数处理) 408 核心算法

非比较型排序：用空间换时间，通过“前缀和”定位绝对坐标，利用“偏移量”处理负数。

🔥 408 考研核心定则 (计数排序)：

  • 时间复杂度：O(n + k) (n是元素个数, k是数值范围max-min+1)
  • 空间复杂度：O(n + k) (需计数数组C和输出数组B)
  • 稳定性：稳定 (Stable) (前提是必须逆序遍历A数组填入B)
  • 负数处理：引入 min 偏移，元素 x 存入 C[x - min]。
速度: 0 / 0
MIN: - MAX: - RANGE: -
1. 输入数组 A (Input)
2. 计数数组 C (Count Array)
↑ 映射关系：索引 index = 真实值 val - MIN
3. 输出数组 B (Output)

void CountingSort(int A[], int n) {
    if (n <= 1) return;
    int min = A[0], max = A[0];
    for (int i = 1; i < n; i++) { // 1. 找最值
        if (A[i] < min) min = A[i];
        if (A[i] > max) max = A[i];
    }
    int range = max - min + 1;
    int* C = (int*)calloc(range, sizeof(int)); // 计数组
    int* B = (int*)malloc(n * sizeof(int));  // 输出组
    for (int i = 0; i < n; i++) C[A[i] - min]++; // 2. 频率统计
    for (int i = 1; i < range; i++) C[i] += C[i - 1]; // 3. 前缀和
    for (int i = n - 1; i >= 0; i--) { // 4. 逆序组装(保证稳定)
        B[C[A[i] - min] - 1] = A[i];
        C[A[i] - min]--;
    }
    for (int i = 0; i < n; i++) A[i] = B[i]; // 5. 写回原数组
    free(C); free(B);
}

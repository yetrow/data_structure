# Part4 学习导航首页 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 将 `part4/index.html` 更新为覆盖当前 12 个演示页的简洁分类学习导航。

**Architecture:** 保持单文件静态页面，HTML 负责三类导航结构，内嵌 CSS 负责响应式卡片和链接交互。所有链接使用相对于首页的实际文件名，不引入脚本、构建工具或外部依赖。

**Tech Stack:** HTML5、CSS3、PowerShell 验证脚本

- 使用纯 HTML 和内嵌 CSS，不增加外部依赖。
- 每个入口只显示算法名称。
- 首页覆盖 `part4` 中除自身外的全部 12 个 HTML 演示页。
- 桌面端使用多列卡片，窄屏自动变为单列。
- 链接必须具有悬停和键盘焦点反馈。

- [ ] **Step 3: 验证链接完整、唯一且均指向现有文件**

Run:

```powershell
$index = Get-Content -Raw -Encoding UTF8 .\part4\index.html
$hrefs = [regex]::Matches($index, 'href="([^"]+\.html)"') | ForEach-Object { $_.Groups[1].Value }
$linkedNames = $hrefs | ForEach-Object { Split-Path $_ -Leaf }
$pageNames = Get-ChildItem .\part4 -Filter *.html | Where-Object Name -ne 'index.html' | Select-Object -ExpandProperty Name
$missing = $hrefs | Where-Object { -not (Test-Path (Join-Path .\part4 $_)) }
$unlinked = $pageNames | Where-Object { $_ -notin $linkedNames }
$duplicates = $linkedNames | Group-Object | Where-Object Count -gt 1
if ($hrefs.Count -ne 12 -or $missing -or $unlinked -or $duplicates) { throw '首页链接验证失败' }
'PASS: 12 个演示页面均有唯一且有效的首页入口'
```

Expected: `PASS: 12 个演示页面均有唯一且有效的首页入口`。

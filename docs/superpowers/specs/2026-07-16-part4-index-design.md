# Part4 学习导航首页设计

## 目标

更新 `part4/index.html`，使它与 `part4` 目录当前的 HTML 演示页面保持一致，作为个人学习时快速进入各演示页的简洁导航。

## 页面结构

首页继续使用响应式分类卡片，并划分为三个区域：

- 字符串：KMP next 数组。
- 树结构：二叉搜索树、AVL 树、红黑树、B 树。
- 排序算法：直接插入排序、冒泡排序、快速排序、简单选择排序、堆排序、归并排序、计数排序。

每个入口只显示算法名称，不添加复杂介绍、搜索、筛选或状态管理。

## 链接规则

所有链接直接引用 `part4` 当前存在的 HTML 文件名，并使用相对于 `index.html` 的 `./文件名.html` 路径。入口按文件编号和学习顺序排列，共覆盖 12 个演示页面，不保留已经删除或改名的旧路径。

## 视觉与交互

- 使用纯 HTML 和内嵌 CSS，不增加外部依赖。
- 保持浅色、清晰、低干扰的视觉风格。
- 桌面端使用多列卡片，窄屏自动变为单列。
- 链接具有明确的悬停和键盘焦点反馈。

## 验证标准

- `part4` 中除首页外的 12 个 HTML 演示页均有且只有一个入口。
- 首页不存在指向旧文件名或不存在文件的链接。
- 分类和显示顺序符合上述页面结构。
- 页面在桌面和窄屏宽度下均可正常阅读与点击。
